# Banker CRM — Phase 4 web desk

Next.js UI for the FastAPI backend (`docs/PHASE4_WEB.md`).

```bash
# API must be running on :8000 (see repo README)
cp .env.example .env.local
npm install
npm run dev
```

Open http://127.0.0.1:3000 and sign in with `CRM_API_USER` / `CRM_API_PASSWORD`.

```bash
npm run build   # production build
npm run lint
```
